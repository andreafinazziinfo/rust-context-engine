## Before you ask

1. [QUICKSTART.md](./QUICKSTART.md) — install + `rtk init` + `rtk doctor`
2. [USER.md](./USER.md) — daily commands
3. `rtk doctor` output (redact secrets)

## Share your savings (Show and tell)

```bash
rtk stats --chart
rtk audit -o rtk-audit.md
rtk --version
```

Post: OS, AI client (Cursor / Claude Code), rough % savings, optional screenshot. No API keys or private repo names if sensitive.

## Maintainer policy (Fase D)

- **No new features** from Discussion votes alone — real-world pain points feed Fase E backlog (max 5 items).
- Bug reports with repro → open an **Issue**.
- Setup questions → **Q&A** Discussion.
